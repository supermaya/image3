# 갤러리별 이미지 자동전환 시간 설정 가이드

## 개요
v2024-11-04-v10부터 갤러리별로 이미지 자동전환 시간을 설정할 수 있습니다.

## 설정 방법

### 방법 1: 크리에이터 관리 페이지에서 설정 (권장) ⭐

1. **크리에이터 관리 페이지 접속**
   - https://pixelplanet-95dd9.web.app/creator-manage.html
   - 크리에이터 계정으로 로그인

2. **갤러리 목록에서 수정**
   - 수정할 갤러리의 "수정" 버튼 클릭
   - "이미지 자동전환 시간 (초)" 필드에 원하는 시간 입력
   - 1~60초 사이 값 입력 (미입력 시 기본값 3초)
   - "저장" 버튼 클릭

3. **즉시 적용**
   - 저장 후 해당 갤러리 재생 시 설정된 시간으로 자동전환

### 방법 2: Firebase Console에서 설정

1. **Firebase Console 접속**
   - https://console.firebase.google.com
   - 프로젝트: `pixelplanet-95dd9` 선택

2. **Firestore Database 이동**
   - 좌측 메뉴에서 "Firestore Database" 클릭
   - `music` 컬렉션 선택

3. **음악 문서 선택**
   - 자동전환 시간을 설정할 갤러리의 문서 ID 클릭

4. **필드 추가**
   - "필드 추가" 버튼 클릭
   - **필드 이름**: `autoplayTiming`
   - **필드 타입**: `number`
   - **값**: 1~60 사이의 숫자 (초 단위)
   - 예: `5` (5초마다 자동 전환)

5. **저장**
   - "업데이트" 버튼 클릭

## 우선순위

자동전환 시간은 다음 우선순위로 적용됩니다:

1. **갤러리별 설정** (`autoplayTiming` 필드)
2. **Portrait 카테고리** (5초 기본값)
3. **사용자 설정** (사용자 정보 탭에서 설정)
4. **기본값** (3초)

## 예시

### 예시 1: 빠른 전환 (2초)
```json
{
  "name": "Fashion Show",
  "category": "Fashion & Beauty",
  "autoplayTiming": 2
}
```

### 예시 2: 느린 전환 (10초)
```json
{
  "name": "Landscape Gallery",
  "category": "Nature",
  "autoplayTiming": 10
}
```

### 예시 3: 설정 없음 (기본값 적용)
```json
{
  "name": "Music Video",
  "category": "Entertainment"
  // autoplayTiming 필드 없음 -> 3초 기본값 적용
}
```

## 콘솔 로그 확인

갤러리 재생 시 브라우저 콘솔(F12)에서 적용된 자동전환 시간을 확인할 수 있습니다:

- `갤러리 설정 자동전환 시간: 5초`
- `Portrait 카테고리 기본값: 5초`
- `사용자 설정 자동전환 시간: 3초`
- `기본 자동전환 시간: 3초`

## 주의사항

- 최소값: 1초
- 최대값: 60초
- 1~60 범위를 벗어나면 기본값(3초)이 적용됩니다
- 숫자가 아닌 값을 입력하면 무시됩니다

## 크리에이터 권장 설정

- **패션/뷰티**: 2-3초 (빠른 전환)
- **포트레이트**: 5-7초 (얼굴 감상)
- **풍경/자연**: 7-10초 (느린 전환)
- **추상/예술**: 5-8초 (중간 속도)
- **액션/스포츠**: 2-3초 (빠른 전환)
